# Fixture provenance

All files in this directory were authored specifically for the RWKV-SRS public
release suite and contain no imported review history or trained model data.

- `reviews.json` is a hand-written synthetic sequence of fictional integer
  identities and review values.
- `expected_outputs.json` was produced by native API version 33 using the
  deterministic model generated by `tests/release/model_fixture.py`.
- The generated Safetensors file is not stored in the repository. Its schema
  is independently enumerated in the generator and its float values are
  derived only from tensor names.

SHA-256 hashes are recorded in `PROVENANCE.json`. These fixtures are intended
for redistribution with RWKV-SRS; their final legal terms follow the project
license selected before the first public release.
